Shopping Cart Java Application

Description
    This is a simple console-based Shopping Cart application developed using Java.The program takes user input for an item name, price, and quantity, then calculates and displays the total cost of the selected item.

Features
* Takes item name as input
* Accepts price and quantity from user
* Calculates total cost automatically
* Displays item name along with total amount
* Uses Scanner for user input

Technologies Used
* Java
* Scanner Class

How to Run
1. Compile the program:
   javac src/ShoppingCart.java
2. Run the program:
   java -cp src ShoppingCart

Sample Output

  Enter the item:
  Apple
  Enter price of item:
  10
  How much quantity:
  2
  Apple Total Amount: $20.0
